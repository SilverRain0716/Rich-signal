## 2. 가격 구조의 장점

### 2.1 사용자 입장

```
1. 진입 장벽 낮음
   - 최저가: 9,900원
   - "일단 시작해보자"

2. 단계적 업그레이드
   - 9,900 → 29,900 → 49,900 → 99,000
   - 필요에 따라 선택 가능

3. 명확한 차별화
   - 각 플랜의 목적 명확
   - "뭘 고를지" 쉬움

4. 번들 할인 강력
   - 기본(9,900) + Signal(9,900) = 19,800
   - 번들가 29,900 → 34% 할인!
   - "번들이 훨씬 낫네?" 느낌

5. 조건식 선택 가능
   - Signal만 원하면 Signal만
   - 조건식도 원하면 번들
   - 유연함
```

### 2.2 당신 입장

```
1. 조건식 보호 가능
   - Signal 사용자는 조건식 몰라도 됨
   - 영업비밀 완벽 유지
   - 특허 안 해도 됨

2. 다층적 수익
   - 기본 플랜 사용자
   - Signal 사용자
   - 번들 사용자
   - 프로/클라우드 사용자
   - 모두 다 수익!

3. 포지셔닝 좋음
   - "당신의 조건식 vs 신호"
   - 신호는 공개해도 됨
   - 조건식은 비밀 유지

4. 업세링 쉬움
   - 9,900 고객 → 29,900
   - 29,900 고객 → 49,900
   - 49,900 고객 → 99,000
   - 자연스러운 단계
```
