## 2. 구독형 수익화 전략

### 2.1 4단계 가격 플랜

| 플랜 | 가격 | 프로그램 | 신호 | 조건식 | AWS | 대상 |
|------|------|---------|------|--------|-----|------|
| **기본** | 15,900 | ✓ | 선택 | HTS만 | ✗ | 초심자 |
| **프리덤** | 29,900 | ✓ | ✓ | 선택 | ✗ | 자유도 추구 |
| **프로** | 49,900 | ✓ | ✓ | 1% 자동 | ✗ | ⭐ 추천 |
| **24시간** | 99,900 | ✓ | ✓ | 1% 자동 | ✓ | 자동화 원함 |

### 2.2 조건식 관리

```
기본 플랜:
  → HTS 조건식만 (사용자가 설정)
  → 당신의 조건식 사용 불가

프리덤 / 프로 / 24시간:
  → 당신의 1% 조건식 (Signal으로 제공)
  → HTS 조건식도 사용 가능
  → AND/OR 조합 가능
```

### 2.3 최대 운용 금액: 5억원

**제약 사항:**
- Kiwoom OpenAPI 성능 제한
- 거래소 규정
- 시스템 안정성 검증 범위

**5억원 초과 시:**
- 첫 경고: 가입 해지 요청
- 미응응: 계약 해지
- 손실 발생: 당사 책임 없음

## 8. 차별화 요소

### 당신의 K-Trader vs 경쟁사

```
경쟁사:
  → 조건식 공개 (분석 가능)
  → 사용자가 복사 가능
  → 차별화 약함

K-Trader:
  ✓ 조건식 비밀 유지 (프로그램에 박혀있음)
  ✓ 신호만 제공 (조건식 분석 불가)
  ✓ 4년 검증된 1% 전략
  ✓ 사용자 조건식도 함께 사용 가능
  ✓ 하이브리드 시스템 (유연성)
  ✓ AWS 완전 포함 (투명성)
```
