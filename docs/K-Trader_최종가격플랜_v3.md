  ✓ 당신의 조건식 선택 가능

    → 둘 다 맞을 때만 신호
    → 가장 보수적
  
  옵션 4: 당신의 조건식 + HTS 조건식 (OR)
    → 하나라도 맞으면 신호
    → 더 많은 신호

  "당신의 조건식도, HTS도, 둘 다도!"
  
사용 방식:
  1. K-Trader 설정에서 모드 선택
  2. 원하는 조건식 조합 선택
  3. 거래 시작
  4. 언제든 변경 가능
  
대상:
  • "자유도 원하는 사람"
  • "당신의 조건식도 시도해보고 싶은 사람"
  • "자신의 HTS 조건식도 쓰고 싶은 사람"
  • "가장 유연한 옵션 원하는 사람"

포함 기능:
  ✓ K-Trader 프로그램 (조건식 선택 가능)
  ✓ Signal 서비스
  ✓ 둘 다 결합 가능
  ✓ 웹 대시보드 (상세)
  ✓ 기본 기술 지원
  ✓ 당신의 조건식 + HTS 조건식
