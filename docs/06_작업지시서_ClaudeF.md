### 포함 항목
1. 서비스 정의: 한국 주식 시장 정보 제공 서비스 (투자 자문 아님)

3. 유료/무료 플랜 구분 및 결제 정책
4. 무료 체험 기간 (1주일) 정책
5. 베타 유저 혜택 및 할인 쿠폰 정책
6. 금지 행위 (다중 계정 생성, 데이터 재배포 등)
7. 서비스 중단/변경 고지
8. 책임 제한 (투자 손실에 대한 면책)
9. 준거법 (대한민국)

### 포함 항목

## Task F-3: 투자 면책 조항 (disclaimer.md)

### 모든 페이지에 삽입할 짧은 버전
```
본 서비스는 투자 자문이 아닌 시장 정보 제공 서비스입니다.
제공되는 정보는 투자 판단의 참고용이며, 투자에 따른 모든 책임은 이용자에게 있습니다.
```

### 수급주 페이지 전용 (상세 버전)
```
당일 수급주는 14:30 기준 잠정치를 바탕으로 산출됩니다.
장 마감 후 실제 수치와 차이가 있을 수 있습니다.
외국인/기관 순매수 데이터는 투자 판단의 참고 자료이며,
해당 종목의 매수를 권유하는 것이 아닙니다.
```

### AI 예상 주도주 전용
```
AI 예상 주도주는 인공지능 분석에 기반한 참고 정보입니다.
실제 시장 결과와 다를 수 있으며, 투자 판단의 최종 책임은 이용자에게 있습니다.
```

## Task F-4: FAQ (faq.md)

### 포함할 질문 목록

**서비스 일반**
- Rich-Signal은 어떤 서비스인가요?
- 투자 자문 서비스인가요?
- 어떤 데이터를 기반으로 하나요?

**주도주/수급주**
- 주도주는 어떤 기준으로 선정되나요?
- AI 예상 주도주는 얼마나 정확한가요?
- 수급주는 무엇인가요?
- 수급주 데이터는 정확한가요? (잠정치 설명)

**플랜/결제**
- 무료로 어떤 기능을 쓸 수 있나요?
- Signal Pro의 혜택은 무엇인가요?
- K-Trader는 무엇인가요?
- 결제 수단은 무엇인가요? (Stripe)
- 구독 취소는 어떻게 하나요?
- 무료 체험 기간은 얼마인가요?

**계정**
- 한 사람이 여러 계정을 만들 수 있나요? (불가)
- 회원 탈퇴는 어떻게 하나요?

**NXT**
- NXT 장외시간 데이터는 무엇인가요?
- 프리마켓/애프터마켓 주도주는 어떤 기준인가요?
